## üèóÔ∏è Technical Infrastructure

### Database Schema Additions Needed:

**friends** table:
- user_id, friend_id, status (pending/accepted/blocked), created_at

**leagues** table:
- id, name, description, creator_id, type, settings (JSON), created_at

**league_members** table:
- league_id, user_id, joined_at, rank, score

**game_modes** table:
- id, name, description, rules (JSON), duration, active

**challenges** table:
- id, type, description, rules (JSON), start_time, end_time, creator_id

**challenge_participants** table:
- challenge_id, user_id, score, rank, completed

**achievements** table:
- id, name, description, icon, rarity, criteria (JSON)

**user_achievements** table:
- user_id, achievement_id, unlocked_at

**notifications** table:
- id, user_id, type, content (JSON), read, created_at

**posts** table (social feed):
- id, user_id, content, likes, created_at

**leaderboards** table:
- id, type, period, data (JSON), updated_at

**messages** table:
- id, sender_id, receiver_id, content, read, created_at

**portfolio_snapshots** table (for historical tracking):
- user_id, total_value, cash, timestamp, stocks (JSON)
